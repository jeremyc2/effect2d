# Scratchpad

Low-stakes **brainstorm** space for Effect2d: half-baked worries, gut reactions, and back-and-forth. Nothing here needs to be defensible in court. The name **less founded** and this folder are the point: vibes over receipts.

## Files

| File | Who edits | Purpose |
| --- | --- | --- |
| [`LESS_FOUNDED_CRITICISM.md`](./LESS_FOUNDED_CRITICISM.md) | **You (human)** | Numbered concerns, questions, and rough criticism. Dump thoughts as they show up. |
| [`LESS_FOUNDED_GUESSWORK.md`](./LESS_FOUNDED_GUESSWORK.md) | **AI only** | Loose replies to items in `LESS_FOUNDED_CRITICISM.md`: agree, disagree, riff, maybe point at code if it helps—but no requirement to prove anything. |
| [`LESS_FOUNDED_REBUTTAL.md`](./LESS_FOUNDED_REBUTTAL.md) | **You (human)** | Your take on what landed in `LESS_FOUNDED_GUESSWORK.md`. The AI does not edit this file. |

## Rules

1. **The human does not edit `LESS_FOUNDED_GUESSWORK.md`.** Only the assistant (or other AI tooling you designate) updates it, usually after you change `LESS_FOUNDED_CRITICISM.md` or ask for a pass.
2. **The AI does not edit `LESS_FOUNDED_CRITICISM.md` or `LESS_FOUNDED_REBUTTAL.md`.** Those are your documents.
3. **Ordering:** You sketch or revise items in `LESS_FOUNDED_CRITICISM.md` → the assistant fills or updates `LESS_FOUNDED_GUESSWORK.md` → you react in `LESS_FOUNDED_REBUTTAL.md` if you want.
